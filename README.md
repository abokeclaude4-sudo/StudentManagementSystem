
# 🎓 Student Management System

A simple Java console application for managing students.

## 🚀 Features
- ➕ Add student
- 📋 View students
- ❌ Delete student
- 🔍 Search student
- 💾 Save students to file
- 🔄 Load students from file


## 🛠️ Technologies Used
- Java
- VS Code
- Git
- GitHub


## ▶️ How to Run

1. Compile the program:

javac StudentManagementSystem.java


2. Run the program:

java StudentManagementSystem


## 📂 File Storage
- Data is saved in:

students.txt

- The file is automatically loaded when the program starts.
