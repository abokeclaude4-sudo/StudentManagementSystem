# Student Management System

A Java-based Student Management System developed to practice core software engineering concepts such as object-oriented programming, file handling, CRUD operations, validation, and GitHub workflow.

## Features

- Add students
- View students
- Search students by ID
- Update student information
- Remove students
- Automatic student ID generation
- Duplicate student prevention
- Automatic name formatting
- File storage using `students.txt`

## Technologies Used

- Java
- VS Code
- Git & GitHub

## Project Structure

- `Main.java` → Main application logic and menu
- `StudentManager.java` → Student management operations
- `students.txt` → Data storage file

## How to Run

1. Open the project in VS Code
2. Run `Main.java`
3. Use the terminal menu to manage students
